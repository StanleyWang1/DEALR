# New XBOX Controller Running on Windows 11

Left Stick Left/Right = Axis 0 (-1 left, 1 right)
Left Stick Up/Down = Axis 1 (-1 up, 1 down)

Right Stick Left/Right = Axis 2 (-1 left, 1 right)
Right Stick Up/Down = Axis 3 (-1 up, 1 down)

Left Trigger = Axis 4 (-1 to 1)
Right Trigger = Axis 5 (-1 to 1)

X Button = Button 2
Y Button = Button 3
A Button = Button 0
B Button = Button 1

Left Bumper = Button 4
Right Bumper = Button 5
DPAD = Hat
